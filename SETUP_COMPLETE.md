### Option 2: Manual Step-by-Step Commands

```bash
# 1. Build web assets
npm run build

# 2. Sync Capacitor
npx cap sync android

# 3. Initialize Git
git init

# 4. Add everything and commit
git add .
git commit -m "Initial commit: Nova AI - Voice Assistant with Android"

# 5. Set main branch
git branch -M main

# 6. Add remote (replace URL with your GitHub repo)
git remote add origin https://github.com/YOUR_USERNAME/nova.git

# 7. Push to GitHub
git push -u origin main
```

### Option 3: One-Liner (If Git Already Initialized)

```bash
npm run build && npx cap sync android && git add . && git commit -m "Update: Nova AI Android ready" && git push origin main
```

## Troubleshooting

### "0 JARs found" Gradle Error
- ✓ **Fixed:** GitHub Actions workflow skips gradle wrapper validation with `-Dorg.gradle.dependency.verification=off`

### Android Build Fails Locally
```bash
# Clear Gradle cache
cd android
./gradlew clean

# Rebuild
./gradlew build
```

### Git Remote Already Exists
```bash
# Remove and re-add
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/nova.git
git push -u origin main
```

### Push Rejected (No Permission)
- Ensure you have GitHub push access
- Generate SSH key or use GitHub personal access token
- For HTTPS: Update credentials in Git credential manager
